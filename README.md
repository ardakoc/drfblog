# drfblog
Basic CRUD blog website project using django-rest-framework

### Installation:
1. Clone the project.
2. Create a virtual environment and install the packages in the requirements.txt file.
3. Migrate.
``` python manage.py migrate ```
4. Run.
``` python manage.py runserver ```
